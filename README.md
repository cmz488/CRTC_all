    就如表格所示，我们来自同一个专业，所学知识高度重合。基于两台机器人的规则，在第一次组会我们决定把我们分成两个小组，分别负责一号车和二号车的开发；同时鉴于我们都有一定的电控知识，我们采取分布式开发，基于ST标准库，将驱动源码分配到每一个人编写和调试。
